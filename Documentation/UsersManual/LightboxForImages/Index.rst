Lightbox for images
^^^^^^^^^^^^^^^^^^^

Create a content element “Images” or “Text & Images” as normal and set
“Enlage on Click”.

.. figure:: ../../Images/UsersManual-LightboxForImages.png

Using images in RTE with activated “Click to enlarge” is a second way
to use Magnific Popup.
